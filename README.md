# hiremeapp

## Setup
    git clone git@github.com:jmsfilipe/hiremeapp.git
    cd src
    npm install
    node server.js
    open http://localhost:8080

    mongodb hostname: ds023714.mlab.com
    mongodb port: 23714
    username: hiremeapp
    database: hiremeapp
    
    node runall.js
    
## Deploy
    hiremeapp.herokuapp.com
